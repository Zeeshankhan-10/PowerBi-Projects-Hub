# 📊 Sales Dashboard Analysis (Power BI Project)

## 📌 Overview
This project presents an interactive Power BI dashboard built to analyze sales performance, track key business metrics, and uncover insights from transactional data. The dashboard enables users to explore trends, compare performance across categories, and support data-driven decision-making.

---
-- <a href="https://github.com/Zeeshankhan-10/PowerBi-Projects-Hub/blob/main/PowerBi%20Project%20-%201.pbix"Dataview </a> 


## 🎯 Objective
- Monitor key sales metrics and KPIs  
- Analyze performance across products and categories  
- Identify trends and patterns in sales data  
- Enable interactive data exploration through dashboards  

---

## 🛠️ Tools Used
- Power BI  
- Data Cleaning & Transformation (Power Query)  
- Data Modeling  
- DAX (Data Analysis Expressions)  
- Data Visualization  

---

## 📊 Dataset
The dataset includes structured business data such as:
- Sales / Revenue  
- Product Categories and Items  
- Quantity Sold  
- Time-based data (Date/Month/Year)  
- Customer or Regional information  

---

## 🔄 Process
- Imported and cleaned raw data using Power Query  
- Transformed and structured data for analysis  
- Built data model with relationships  
- Created calculated measures using DAX  
- Designed interactive dashboards with filters and visuals  

---

## 📈 Key Features
- Interactive filters (slicers) for dynamic analysis  
- KPI cards to track overall performance  
- Trend analysis using line and bar charts  
- Category and product-level breakdowns  
- User-friendly dashboard layout for easy navigation  

---

## 📈 Key Insights
- Identified top-performing products and categories  
- Observed trends in sales over time  
- Highlighted variations in performance across segments  
- Enabled quick comparison of key business metrics  

---

## ✅ Conclusion
This dashboard provides a clear and interactive view of business performance, allowing stakeholders to quickly identify trends, monitor KPIs, and make informed decisions. It demonstrates the use of Power BI for transforming raw data into meaningful insights.
