# 🧠 Teen Mental Health & Social Media Analysis (Power BI Project)

## 📌 Overview
This project presents an interactive Power BI dashboard analyzing the relationship between social media usage and teen mental health. It explores how factors such as screen time, platform usage, and demographics influence stress, anxiety, and addiction levels.

---

## 🎯 Objective
- Analyze the impact of social media usage on teen mental health  
- Compare platform usage across genders  
- Evaluate key mental health indicators such as stress, anxiety, and addiction  
- Identify behavioral patterns related to screen time and sleep

## 🛠️ Tools & Technologies
- Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Data Visualization  

---

## 📊 Dataset
The dataset includes information related to:
- Age groups (13–19)  
- Gender  
- Social media platform usage (Instagram, TikTok, etc.)  
- Daily screen time  
- Sleep duration  
- Stress, anxiety, and addiction levels  
- Physical activity  

---

## 🔄 Process
- Cleaned and transformed raw data using Power Query  
- Structured data for analysis and visualization  
- Created calculated measures for averages (screen time, sleep, stress, etc.)  
- Designed an interactive dashboard with multiple visuals and KPIs  

---

## 📈 Dashboard Features
- **Platform Usage by Gender** (stacked bar chart)  
- **Age Distribution** (treemap visualization)  
- **KPI Cards** for:
  - Average screen time before sleep (1.76 hrs)  
  - Average sleep hours (6.43 hrs)  
- **Mental Health Indicators**:
  - Stress Level (5.45 / 10)  
  - Anxiety Level (5.64 / 10)  
  - Addiction Level (5.57 / 10)  
- **Daily Social Media Usage** (donut chart by gender)  
- **Physical Activity Comparison by Gender** (bar chart)  

---

## 📈 Key Insights
- Teens spend an average of **1.76 hours on screens before sleep**, which may impact sleep quality  
- Average sleep duration (**6.43 hours**) is lower than recommended levels  
- Moderate levels of **stress, anxiety, and addiction** are observed across users  
- Social media usage patterns differ slightly between male and female users  
- Physical activity levels remain relatively similar across genders  

---

## ✅ Conclusion
This dashboard highlights the potential relationship between increased social media usage and mental health challenges among teenagers. The insights can help educators, parents, and policymakers better understand behavioral patterns and promote healthier digital habits.
